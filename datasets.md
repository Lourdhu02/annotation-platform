# Public Multi-Modal Annotation Datasets

1. **COCO** – 330K images, 80 object categories, captions,
   keypoints, panoptic seg. JSON format. cocodataset.org

2. **Localized Narratives** – 849K images (COCO + Flickr30k +
   ADE20K + Open Images). Voice + mouse trace + caption per
   image. JSONL + OGG. google.github.io/localized-narratives

3. **MultiModalQA** – 29.9K QA pairs over text + tables + images.
   JSONL format. github.com/allenai/multimodalqa

4. **ShareGPT-4o** – 200K images + 10K videos + 10K audio with
   GPT-4o captions. JSONL. sharegpt4o.github.io

5. **MMDocRAG** – 8K+ document QA pairs with text + image quotes,
   interleaved answers. JSONL. github.com/MMDocRAG/MMDocRAG

6. **FLARE** – 87.7K video clips (225h total) with vision-only,
   audio-only, and unified captions + 274K user queries. JSONL.
   huggingface.co/YqjMartin/FLARE

7. **Trivenika (Coalescence)** – 383K entries combining 31 source
   datasets across image, audio, text. huggingface.co

8. **Cole Conversational Corpus** – 102K+ turns (text, image,
   audio, video, docs) from real production AI assistant. JSONL.
